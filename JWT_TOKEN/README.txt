Jwt Token Creation and Validation

jwt token three parts
1.Header
2.Payload
3.Signature

Header - It consists of token's type.
Payload - It consists of claims to set the Payload the payload contains authorization, expiration and user.
Signature - It consists of Header,Payload and Securitykey.

In create Token Function we are creating token.
In Valid Token Function we are validating the token.
